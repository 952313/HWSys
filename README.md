# 📚 作业管理平台

一个基于 Flask + SQLite 的学校作业管理系统。

---

## 🚀 快速开始

### 环境要求
- 见requirements.txt

### 安装运行

```bash
# 1. 克隆或下载项目
cd assignment-platform

# 2. 安装依赖
pip install -r requirements.txt

# 3. 运行
python main.py

# 4. 访问
# 浏览器打开 http://localhost:5000
```
